================================================================================
                        容量表合成工具 - 操作说明
================================================================================

【一、目录结构说明】

CapPhysCombine/
├── CapPhysCombine.exe    # 主程序（双击运行）
├── data/                 # 数据文件目录（需要放入源数据Excel）
├── logs/                 # 日志目录（自动生成）
└── *.dll                 # 运行依赖文件（勿删）


【二、数据文件要求】

请将以下Excel文件放入 data/ 目录：

┌─────────────────────────────────────────────────────────────┐
│ 文件名格式                          │ 说明                  │
├─────────────────────────────────────────────────────────────┤
│ 5G小区容量-周*.xlsx                 │ 5G小区周容量数据      │
│ 5G小区容量报表*.xlsx                │ 5G小区日容量报表      │
│ 5GMR覆盖-小区天*.xlsx               │ 5G MR覆盖数据         │
│ 5G小区性能KPI报表*.xlsx             │ 5G KPI性能数据        │
│ 重要场景-周*.xlsx                   │ 4G重要场景周数据      │
│ 重要场景-天*.xlsx                   │ 4G重要场景日数据      │
│ 4GMR覆盖-小区天*.xlsx               │ 4G MR覆盖数据         │
│ 共站同覆盖小区_4g_5g.xlsx           │ 4G/5G共站覆盖关系     │
└─────────────────────────────────────────────────────────────┘

注意：
- 文件名中的 * 表示通配符，可以匹配任意字符（如日期）
- 系统会自动选择最新的文件进行处理
- 每个类型至少需要1个文件，否则程序会报错


【三、运行步骤】

1. 准备数据
   将上述Excel文件复制到 data/ 目录下

2. 运行程序
   双击 CapPhysCombine.exe 启动程序

3. 开始处理
   点击"开始生成"按钮，等待处理完成

4. 查看结果
   处理完成后，会在程序同级目录生成：
   - 合成_容量表_5G.xlsx
   - 合成_容量表_4G.xlsx
   - 容量表_45G.xlsx


【四、常见问题】

Q1: 提示"未找到匹配文件"
A: 请检查 data/ 目录下是否有对应格式的Excel文件，文件名是否符合上述格式

Q2: 程序无法启动，提示缺少DLL
A: 请确保所有.dll文件与exe在同一目录，不要单独移动exe文件

Q3: 处理过程中报错
A: 请检查日志文件 logs/app_YYYYMMDD.log 查看详细错误信息


【五、技术支持】

如有问题，请查看日志文件或联系技术支持。
